5 The project has four targets: a Cocoa framework with its unit test bundle, an
6 iOS static library with its unit test bundle. The framework and the library
7 share the same source files. The sources use Automatic Reference Counting (ARC).
8 
9 Why the plural project name? The project goes by the name 'Core Foundation
10 Sockets' rather than 'Core Foundation Socket.' This avoids confusion with
11 `CFSocket` which represents just one component of the CFSockets project. The
12 project's monolithic header imports as `<CFSockets/CFSockets.h>`. The
13 additional _s_ for the project prevents a clash with header
14 `<CFSockets/CFSocket.h>` which only imports the `CFSocket` class.
